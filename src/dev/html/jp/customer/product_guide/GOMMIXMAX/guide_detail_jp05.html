ショートカットキー

(1) 一般

一般
新しいプロジェクト Ctrl + N コピー Ctrl + C
プロジェクトを開く Ctrl + O 貼り付け(上書き) Ctrl + V
メディアクリップを
読み込む		 Ctrl + I 貼り付け(挿入) Ctrl + Shift + V
保存 Ctrl + S 複製 Ctrl + D
名前を付けて保存 Ctrl + Shift + S 削除 Delete
プロジェクトを全て 保存 Ctrl + Shift + E 書き出し Ctrl + E
元に戻す Ctrl + Z プログラム終了 Alt + F4
やり直す Ctrl + Y プログラム情報 F1
トリミング Ctrl + X 環境設定 F5

(2) タイムライン

タイムライン
クリップを追加 (タイムライン) Q マグネティック ON/OFF Alt + P
選択されたクリップの 後に追加(挿入) W 再生ヘッドにクリップを プッシュ Ctrl + Right(→)
切った後クリップ追加 (挿入) E 再生ヘッドにクリップを プル Ctrl + Left(←)
切った後クリップ追加 (上書き) R ビデオトラックを追加 Shift + V
リップル削除 Ctrl+Delete オーディオトラックを追加 Shift + A
クリップ分割 Ctrl + K トラックトランジション拡張 ON/OFF Ctrl + Q
倍速/長さ変更 Alt + R トラックトランジション 全て拡張ON/OFF Ctrl + Shift + Q
オーディオ調整 Alt + D トラック拡張 ON/OFF M
エフェクト/オーバーレイ 属性 Alt + S トラック全て 拡張ON/OFF Ctrl + Shift + M
属性編集 (テキスト/オブジェクト) Alt + P マーカー編集 Shift + M
トラックトランジション 属性 Alt + T 前のマーカーに移動 Shift + Left(←)
クリップトランジション 属性 Alt + C 次のマーカーに移動 Shift + Right(→)

(3) プレビュー

プレビュー
再生/一時停止 Space 前のクリップ Alt + Left(←)
停止 Ctrl + Space 次のクリップ Alt + Right(→)
前のフレーム Left(←) 開始点 Home
次のフレーム Right(→) 終了点 End

機能ごとにショートカットキーを設定するショートカットキータブです。
全て解除をクリックするとユーザーがショットカットキーを設定できます。
