Edit Sounds

Select Edit Channel

You can select the sound output channel (side, left, right) and edit the sound in three dimensions.

Area Selection

In the waveform area of the sound, you can intuitively select and adjust the desired area by long-press drag, and you can select the top start section, end section, and length to make detailed selection.

Cut

You can select the part you want and remove only that part or you can easily remove the area that is not selected.

Fade in/out

You can select the area you want to edit and create a pop-up when you click the bottom fade in/out button.
In a fade-in pop-up, you can set the start and end points and make the sound gradually louder.
In the fade-out pop-up, you can set the start and end points, and gradually reduce the sound.
