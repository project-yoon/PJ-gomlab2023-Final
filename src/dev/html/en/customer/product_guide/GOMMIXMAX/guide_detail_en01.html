Product Description

1. GOMMix Max

GOM Mix Max is an easy-to-use video editing program that can be used to turn your video, photo, and audio clips into a unique video by applying various effects that can express your own style.

It offers a simple interface and provides with a huge selection of styles. With a simple introduction to the program, you can make videos into the way you want it. Using various effects provided for each category, you can create a dynamic video with pictures. Also, you can easily create Thumbnails by exporting still cuts of the video you have created.

2. Main Features

(1) Multi-track video editing

  • Easy configuration of clip layouts by providing multiple video tracks and audio tracks
  • Supports video hiding in each track, mute, track locking, etc.

(2) Practical effects such as color and image correction, chroma-key, masking, etc.

  • Color balance, level, vibrance, tint, skin tone, duotone, and other various color effects
  • Adjust image effects such as blur, bloom, out-focus, grunge, etc.
  • Coloring with LUT color filters
  • Split Screen function using screen cutting presets
  • Alpha channels such as Chroma-Key, thumbnail border, shape mask blur, shape mask/crop, etc.
  • Shape drawing animation supporting trim path
  • Screen distortion effects such as Reshape 4 Points, Square Zoom, Waving Flag
  • Keyframe volume control function

(3) Motion graphic feature using keyframes

  • Animation to set frame position, size, rotate, opacity, and speed graphs
  • Camera walk and 3D rotate using perspective effects

(4) Programmable overlay, background, and object clip using GPU

  • Easy change for users’ preference by providing preset clips made from the programs
  • Background clip decoration used as a wallpaper of video
  • Provides object preset as decorative clip
  • Video decoration with overlay clips such as vignette, particle, bokeh, etc.
  • Provides clip to clip transitions, blending, and track matte

(5) Convenient editing environment

  • Intuitively understanding the function through effect preview
  • Clip editing suitable for the situation such as insert, overwrite, cut and insert, cut and overlay, pull and delete, delete space, etc.
  • Edit control input using mouse sliding and wheel

(6) Other Features

  • Small and light-weighted program
  • Similar concept to professional video editing methods. Provide continuity in learning when switching to professional tools
