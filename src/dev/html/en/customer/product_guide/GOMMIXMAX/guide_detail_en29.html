Source and Effects Panel

Sticker Tab

A useful tool in video production, sticker, is provided as independent clips.

After selecting a desired text in the sticker preset, drag it to the timeline. Edit its properties to change up the style, then apply effects. As an example, the “Emoji 01” object clip is as follows.

After changing the size, location, and other variable properties, clicking on “OK” will apply these changes.

You may save this new text style as a preset to conveniently use it again later.

1. Sticker Property Edit

Sticker Property Edit
It is the coordinate for the left top position of the sticker frame.
The position coordinate is in the range of (0, 0) to (100, 100).
At the left top corner of the video screen, the corresponding coordinate is (0.0) and at the right bottom, it is (100, 100). Unit is in '%'.
Move the sticker frame to one of the 9 areas set ahead of time.
This is the size of sticker frame. Enter the desired font size on your own or place the mouse pointer on edit control and hold it down and move it left and right to change the font size.
[Maintain Original Ratio] If the checkbox is enabled, it maintains original ratio when you adjust the scale of the sticker. If the checkbox is disabled, you can freely adjust the width and height of the sticker.
[Scale by Center Point] If the checkbox is enabled, it sets the center as the anchor point. If the checkbox is disabled, it sets the left top as the anchor point.
Rotate Rotate a sticker frame. If you want to rotate it clockwise, use the [+] angle. If you want to rotate it counterclockwise, use the [–] angle.
Opacity Set the level of opacity for the sticker frame.
[Repeat Image] If the checkbox is enabled, it allows the sticker clips to replay their animations as long as they are on the timeline. If the checkbox is disabled, the sticker clip's animation will play only once, and even if you extend the clip's duration, it will stop at the last frame.

This is an animation type for sticker frame appearing. Graph function is used to set the motion graph for the sticker animation. Set the time of appearing. Check [Fade In] to make them appear slowly.

This is an animation type for sticker frame disappearing. Graph function is used to set the motion graph for the sticker animation. This is the time of disappearing. Check [Fade Out] to make them disappear slowly.

Sticker Property Edit
In this mode, you can see the progress of sticker frame animation in each segment of the timeline. When the effect related to sticker animation or movement is set, you can move the playback head of the monitor to see the animation status of each frame.
In this mode, you can see the editing details of the sticker clip.
Using the preset provided, you can create and use your own presets. When a user clicks Save Preset, the entire text item that was set at the top will be saved as a preset. You can check the saved preset in Sticker > My Preset menu.
