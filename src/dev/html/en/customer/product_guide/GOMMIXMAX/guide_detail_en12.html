Top Bar Menu Panel

1. Home

Home
Home Moves to the Home Screen

2. File

File
New Project Makes a new project file. It is made in the folder designated in the file location of the setting. The extension is “.gmxp”(GOMMix Project file)
Open Project Uploads the saved project file (.gmxp)
Open Recent Projects Quickly uploads the recent project files you have worked on
Save Saves the project file with a different name
Save As Saves the project file with a different name
Save All Projects Saves by copying clips of different paths into the designated folder
Project Settings Sets the resolution and number of frames for the new project file
Settings Chooses the file path, language, editing environment of the program
Export Exports (encoding) the edited project file into a video
Export Current Frame Exports the screen of the clip the play head is at in image file
Export Selected Clips Exports selected clips on the timeline track
Open Export Folder Opens the folder where the exported videos are saved
Exit Exits the program

3. Edit

Edit
Undo Undoes what you just did
Redo Redoes what you just did
Slice Clip Cuts the selected source clip and text in the list window of the project clip and the selected clip in the timeline work area to store it in the internal storage
Copy Copies a clip or the selected item into the internal storage
Paste Pastes a clip or item stored in the internal storage
Duplicate Duplicates a text, background, or object clip made in the project clip list
Delete Deletes the clip registered in the project clip list, or deletes by making the chosen clip in the work area go blank
Ripple Delete Deletes by dragging a selected clip without any space in the timeline work area
To add/edit Markers Adds markers on the timeline to indicate the position or time of the clips.

4. Clip

Clip
Import Media Clip Uploads a media clip into the project clip list
Create Motion Image Clip Makes a clip after setting the appear/disappear motion and the size of the image clip
Speed Changes the length of the clip
Effects/Overlay Properties Views all the effects/overlay added in a selected clip in the timeline work area
Delete All Effects/Overlay Deletes all the effects/overlay added in selected clips in the timeline work area
Duration Changes the length of the clip
Edit Properties Edit the clip properties of the Text/Object
Audio Adjustment Changes the settings of a selected audio clip in the timeline work area
Clip Transition Properties Display the properties of a selected clip transition control clip in the timeline work area
Track Transition Properties Views the properties of a selected track transition control clip in the timeline work area

5. View

View
Project/Preview Switches the location of the Source and Effects Panel and Preview Panel
Reset Layout Resets the size of the window and layout window

6. Help

Help
Video Guide Shows the GOM Mix video guide.
User Guide Shows the online Help.
Tutorial Guide Shows the basic usage of the program.
FAQ Display the FAQ
Question/Suggestion You can move to the GOMLab page to submit complaints, improvements,
1:1 inquiry, and suggestions
Login Shows the login window for non-paying users
Coupon Registration Shows the coupon registration window for non-paying users.
License Information Shows the license information. In case the program is without the license,
a pop-up window will show up to license the program.
About GOM Mix Shows the program information.
