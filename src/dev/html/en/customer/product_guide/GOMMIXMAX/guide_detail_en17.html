Source and Effects Panel

Overlay Tab

This shows the decorative clips that could be overlapped with other clips without any change of propertie in the video. After selecting a clip in the timeline, you can add several different overlays in the same clip. The same overlay cannot be added in a clip more than once.
If you wish to do so, however, add a 'Solid Color' background clip in the timeline and change it to become transparent. Then add the overlay you wish to appear once more.

For example, if you select Confetti overlays, then the Overlay Clip Property Setting will appear.

Change direction, amount/size, speed, and other variables then press the [OK] button to apply an overlay.

While an overlay is applied, apply [Transform] effects to make sure the overlay is properly applied to the clip with a changed size. If the order is switched, then go to the [Effect/Overlay properties] and change the order of effects and overlay.
