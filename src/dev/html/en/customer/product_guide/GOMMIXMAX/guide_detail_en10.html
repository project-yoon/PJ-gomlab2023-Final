Function Learning

Trim

Trimming refers to the adjustment of the start and end time for the selected clip.

There is a fixed duration for a video clip or audio clip. You can cut this clip or place the mouse pointer at the end or front of the clip to adjust the duration of the clip. But since the duration is already fixed, you cannot increase the length of the clip but can reduce it. If a clip is already trimmed, then you can increase the duration to the original.

Unlike video or audio clips, you can extend the duration of text, background and object clips. The extended time will be the clip duration and if you reduce the length, it means trimming. It is initially created with the length set in the environment setting and if the length in the timeline is reduced, trimming will be applied. If you increase it beyond the initial length, then the extended time without trimming will be the clip duration.

In the [Speed] menu, you can set the speed.
