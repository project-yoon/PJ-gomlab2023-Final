Home Screen

Home screen is the first page of GOM Mix you will see when you start the program.

1. New Project

Once you click on the New Project button, a separate pop-up window of the project setting will show up.

New Project
Name This is the name of the project file that will be used temporarily when you are making a new project file.
You can change the name of the project as you wish.
File Path This is the location of the folder the project file will be saved. Click on the and change the folder.
Aspect Ratio You can set the width and height of the video.
Generally, the landscape video has an aspect ratio of 16:9 and the portrait video has an aspect ratio of 9:16.
If you want to use the ratio used at specific platforms, select the aspect ratio and the resolution.
Resolution If your desired ratio is not an option, you can directly enter the resolution after choosing the User Setting.
You will see the list of the video resolution of the selected aspect ratio, and you can select the resolution here.
Frame Rate If your desired resolution is not an option, you can directly enter the resolution after choosing the User Setting.
Choose the number of frames made every second. (60 frames max)

2. Open Project

Once you click on the Open Project button, a file explorer will be opened for you to bring the project file you were working on.

3. Recent Projects

This list gives you a list view of the projects you have recently worked on. The one with the latest revision is displayed at the top, and clicking on the project will direct you to its corresponding screen.

4. Video Tutorial

Move to the YouTube channel to watch the GOM Mix tutorial video. Watch the video to learn basic editing methods

5. User Guide

Click the User Guide button to move to the user guide web page.

6. FAQ

Click on the FAQ button to go to the FAQ page of GOM Lab website.
