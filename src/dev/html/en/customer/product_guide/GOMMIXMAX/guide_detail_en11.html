Function Learning

Markers

1. Add & Delete Markers

If you add markers, you can mark the important time point and easily find the clip.

How to add & Delete Timeline Marker
  • 1) Move the Playhead to the desired position.
  • 2) Click [Add/Edit Marker] in Timeline tool bar or right-click on Timeline ruler > Click [Add/Edit Marker]. A Marker is added to the Timeline ruler.

When the marker is adjusted, the Playhead moves to the same position. Marker cannot be located in the same location. For example, if you move Marker A to B, Marker B is deleted. To delete a marker, select a marker and press the [Delete] key or right-click on the Timeline ruler and select [Delete Marker].

2. Edit Markers

Double-click the marker, and you can edit marker. You can set markers such as name, color, description, etc.
If you click [Prev], [Next] in Edit Marker, you can go to the edit window of the previous or next marker to quickly find the marker information. Clicking [Prev] moves to the previous marker, and clicking [Next] moves to the next marker.

Edit Markers
Name Enter the name of the marker.
Color You can select the color of the marker.
Time Indicates the detailed time at which the marker is located.
Description You can write description or note about a selected part of the timeline.

3. Move Markers

  • How to move between markers: Right-click, and you can move to the Next / Previous marker.
  • Move marker: After selecting a marker, you can drag it in the Timeline ruler to move it. When you move the marker to a position in front or behind of a clip in the timeline, you can see the marker guide, and you can easily move it to the desired location.
