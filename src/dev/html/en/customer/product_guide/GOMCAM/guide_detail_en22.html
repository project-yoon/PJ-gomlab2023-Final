Image Editor

Start Image Editor

The image editor is a feature that allows you to edit the images captured by GOM Cam using drawing tools or cropping tools.

Try launching the image editor in the following order.

1. Start From Home

After taking a screen capture, hover your mouse over the thumbnail of the image file in the explorer area of the home screen, then a menu will appear. Click the [Edit Image ] button in the menu to launch the image editor

2. Start with Quick Cacpture

Below shows how to execute via clicking [Capture Complete Pop-up].

  • ① On the home screen, check 'Automatically run at Windows startup' and 'Minimize to system tray when programs are closed' in the [Settings] menu > More in [General].
  • ② Right-click the [GOM Cam ] icon in the system tray and select the Capture menu that appears or use the shortcut key to proceed.
  • ③ When you complete a capture, a complete capture toast pop-up appears, and clicking the complete capture pop-up will automatically launch the image editor.
  • ④ Simply edit the captured image in the image editor.
