Settings

1. General

Settings - General
Language Setting You can set the language of the program. GOM Cam supports Korean/English/Japanese.
Set file path You can set the folder path of the files that will be saved after recording and image capture.
Save Format You can choose the file format that will be saved after image capture.
Others Other basic settings can be selected and changed.
Reset All Resets all the settings you changed to the initial settings.
Close It exits the settings pop-up.

2. Device

Settings - Device
Video Settings

You can change and adjust the settings for the video when screen recording.

  • Frames/second: Allows you to set the frame rate of the video.
    (Even if you set frames/second, the actual recorded frame rate may vary depending on your PC's performance.)
  • Compression Format: You can set the video compression format.
    (H.264 (CPU) uses only the CPU for encoding, so high CPU utilization can cause screen stuttering in recorded files.)
  • Size: You can set the size of the video.
Audio Settings

You can set settings for audio when screen recording.

  • Microphone: You can choose microphone device connected to your PC.
  • Audio: You can choose the audio device connected to your PC.
Others When recording, you can choose the video/audio quality. Depending on the quality, the size of the file may vary.

3. Recording

Settings - Recording
Mouse Pointer Recording Set whether to use the mouse pointer when recording the screen. You can choose whether to use the mouse pointer, the left mouse button, or the right mouse button, and you can set the color when checked. Mouse pointer recording only applies to screen recording.
Automatic Shutdown

When recording a screen, you can set the automatic shutdown by specifying the recording time or recording capacity.

  • Recording Time: Automatically shut down after recording for a set amount of time. The maximum recording time you can set is 23 hours, 59 minutes, and 59 seconds.
  • Recording capacity: Automatically shut down after recording up to the recording capacity you set. The maximum recording capacity you can set is 20GB.
  • Select Action: You can choose to end recording, end program, or shut down your PC.
Split Save

When screen recording, you can set the split save by specifying the recording time or recording capacity.

  • Recording time: Record for a selected amount of time and then automatically split save. You can choose from 1 minute to 120 minutes. The maximum recording time that can be specified by direct input is 23 hours and 59 minutes.
  • Recording capacity: Record up to the selected capacity and automatically split save. You can choose from 25 MB to 20 GB. The maximum recording capacity that can be specified with direct input is 20GB.
Draw You can set the font when adding text in the drawing tool.
Others You can change the default settings for other recordings.

4. Shortcuts

It provides hotkeys to help you work quickly on each function.

Settings - Shortcuts
Hotkey Entry Hotkey items and currently set hotkeys are exposed.
Hotkey Settings You can change the hotkeys for the selected items, and all shortcuts will be turned off when you click the [Turn All] button.
