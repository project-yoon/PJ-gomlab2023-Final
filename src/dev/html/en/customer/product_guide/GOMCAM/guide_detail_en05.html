Explorer Area

1. Function Menu

Function Menu
View You can change the way you view a saved file. You can change the view in the following order: large thumbnail view, small thumbnail view, and list view.
Filter Filters can be collected according to the saved file type. You can choose from Full View/Video/Audio to collect and view or find the file types you want.
Ascending Descending You can sort and change the files in ascending or descending order.
Sort by You can check the files by changing them according to the file sort criteria. Check the file by changing it according to the file name/size/creation date/playback time.
Search for Tags
 After adding a specific tag to a file, you can search for it in the search box. Search for the tags you have added to quickly find the file you are looking for.
Open Save Folder You can open and check the saved folder location of the selected file.

2. Video File

When you hover over the mouse on a file thumbnail, buttons for various functions will appear.

Video File functions
Extract Audio It allows you to extract audio from video files to MP3 files. The extracted audio files will be saved in the GOM Cam 2024 > ExtractAudio folder.
Video Editing You can select the editing mode and launch the video editor to edit the recorded video.
Share You can upload the selected files to YouTube/Google Drive/Microsoft OneDrive/Vimeo.
Playback Play the selected video file with the video player. After selecting the [More] button, you can click the [Playback] menu or double-click the file to play the video in the player.
File Information It is a menu where you can check the information about the file. You can edit the file name and tags, check the file location, creation date, size/resolution, and duration/frame rate. The following can be checked in the pop-up that appears after selecting the [File Information] menu from the [More] button.
Open Folder You can open the folder where the file is stored.
Delete Delete the selected file.
File Name You can modify the file name by hovering over the mouse on the filename and right-clicking.
Tag You can add or modify tags by right-clicking the mouse in the tag area under the tag file name. You can use the [Search Tag] function to search and find files with the corresponding tag.

3. Image File

Image File - More
Image Editing You can launch the image editor to edit the captured image.
Share You can upload the selected files to YouTube/Google Drive/Microsoft OneDrive/Vimeo.
Open Open the selected image file to check it with the image viewer. You can open it with the image viewer by clicking the [Open] menu after clicking the [More] menu, or by double-clicking the file.
File Information The File Information pop-up appears, allowing you to edit the file name and tags, as well as the file location, creation date, and size/resolution.
Open Folder You can open the folder where the file is stored.
Delete Delete the selected file.
File Name You can modify the file name by hovering over the mouse on the filename and right-clicking.
Tag You can add or modify tags by right-clicking the mouse in the tag area under the tag file name. You can use the [Search Tag] function to search and find files with the corresponding tag.
