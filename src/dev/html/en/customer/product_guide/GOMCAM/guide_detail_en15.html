GIF Animation

GIF animation is a feature that combines a series of images into a single file and turns them into an animated image. You can use the GIF animation feature to create an image file that moves parts of your video.

1. Toolbar Before Capture

Toolbar Before Capture
Start Capture You can start a capture.
Reset Area It resets the capture area. After clicking the Reset Area icon, you can drag the mouse to reset the capture area.
GIF Animation Capture Settings
 When you click it, the GIF Animation Capture Settings pop-up will appear and you can change the FPS and speed.
Close Close the Capture toolbar and the home screen will appear.

2. Toolbar While Capturing

Toolbar While Capturing
Pause
Resume 		You can pause the resume capture. During a pause, the [Pause] icon changes to the [Resume] icon and clicking the icon resumes recording.
End Capture The capture ends and the GIF Preview pop-up appears. You can check and edit the captured GIF animation in the pop-up.
Capture Time You can check the captured time in real-time.
Close Close the Capture Toolbar and the home screen will appear. When clicking the icon during capture, the [GIF Preview] pop-up will appear, allowing you to view and edit the captured GIF animation.

3. GIF Animation Capture

Follow the steps below to capture GIF animations.

  • ① On the home screen, click the [More ] button in the upper right corner of the [Image Capture] button and click the [GIF Animation] menu.
  • ② The Capture Toolbar will appear. The toolbar allows you to start capture, reset area, and set capture settings.
  • ③ Click the [Start Capture ] button to start capturing GIF animations.

    Click the [GIF Animation Capture Settings ] button to set the FPS and speed.

  • ④ When the capture is finished, click the [End Capture ] button. The capture will end and the GIF Preview pop-up will appear and you can check the saved GIF animation file.
  • ⑤ Click the [Edit] button in the [GIF Preview] pop-up and the [Edit GIF] pop-up will appear. You can simply edit the saved GIF animation file via the edit pop-up. After editing, it will be saved as a GIF animation file when exported.
  • ⑥ Click the [Close] button to end the capture and the home screen will appear. You can check the GIF animation file in the explorer area.
