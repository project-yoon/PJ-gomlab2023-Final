Set Up File Naming Rules

Here's how to set up file naming rules:

  • 1. Click the [Settings] button on the main screen.
  • 2. When the Settings window pops up, you can select File Rules.
  • 3. Click the [Advanced] button and you can directly add or delete the desired file name in the window that will be created.
