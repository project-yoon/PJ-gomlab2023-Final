Setting Preferences

Playback 360 VR Video

Move 360︒ VR video left or right directions with a keyboard or mouse. On the touch screen, you can also use touch gestures.

If you click “Playback 360︒ VR Video”() icon, a pop-up menu will appear.

Playback 360 VR Video

Playback 360 VR Video
Playback 360︒VR mode Start VR mode. If you start VR mode, it will turn into 360︒VR Mode stop menu, and if you select it again, the VR mode will stop.
Import 360︒ Videos

Able to import 360︒video files.

  • Open 360︒video: Open a 360︒video file in your PC.
VR mode Settings

Able to set up a type of videos to be used in a VR mode.

  • Mono: Select if a file is a general 2D video.
  • Left/right video: Select if a file is a 3D video split horizontally.
  • Up/down video: Select if a file is a 3D video split vertically.
VR Mapping Setting

You can set up a mapping when converting a 3D image to be used in VR mode into a 2D image.

  • Equirectangular: When converting a 3D image into a 2D image, map the longitude by changing the x-axis and the latitude by the y-axis.
  • Equi-Angluar Cubemap 3x2: When converting a 3D image into a 2D image, map six squares into one plane. 3x2 layouts are currently used as the default.
