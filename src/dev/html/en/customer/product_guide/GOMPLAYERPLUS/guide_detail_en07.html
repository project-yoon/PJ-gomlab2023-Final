Playing video

How to Play VR Video

Able to play videos produced in a VR (Virtual Reality) format. If you select the Playback 360︒video in the basic menu, you can import and play VR videos in the pop-up menu displayed.

Playback 360︒VR video menu

Playback 360︒VR video menu
360︒VR Mode Setting Start or stop VR mode.
VR video Setting

Set the types of videos to be used in VR mode.

  • Mono: Select if it is a General 2D video.
  • Left/Right Mode: Select if a file is a 3D video split horizontally.
  • Up/Down Mode: Select if a file is a 3D video split vertically.
VR Mapping Setting

You can set up a mapping when converting a 3D image to be used in VR mode into a 2D image.

  • Equirectangular: When converting a 3D image into a 2D image, map the longitude by changing the x-axis and the latitude by the y-axis.
  • Equi-Angluar Cubemap 3x2: When converting a 3D image into a 2D image, map six squares into one plane. 3x2 layouts are currently used as the default.
