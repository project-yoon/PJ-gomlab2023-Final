Basic CSS Rule

CSS 작성 규칙

  • sass 문법이 아닌 scss문법으로 작성
  • 파일명 : 메뉴명을 영문으로 작명 예: 고객센터 > customer.scss
  • 공통 내용 scss파일을 제외하고, 각 메뉴별(고객센터, 회원 등)로 분리
  • class(id)명은 소문자, 단어 조합은 하이픈(-)으로 구분 권장 예: btn-close.png등
  • 컴포넌트별로 작성할 것
  • 섹션 구분을 위한 주석 상위에서 줄내림 할 것
  • 의미 없는 줄내림 하지 않습니다

4 원하는 사이즈가 맵에 없는 경우,
기본값으로 설정되도록 세팅		 @mixin txt($size)

_variables.scss		 

	@mixin txt($size) {
		@if map-has-key($typography-styles, $size) {
			$values: map-get($typography-styles, $size);
			@include apply-typography-style(nth($values, 1), nth($values, 2));
		} @else {
			// $size가 맵에 없는 경우 기본값으로 16 적용
			$values: map-get($typography-styles, 16);
			@include apply-typography-style(nth($values, 1), nth($values, 2));
		}
	}
5 실제 사용 예시 @include txt(폰트사이즈);

*.scss		 

	span {
		@include txt(24);
	}

✅ 실 사용 예시

	@include txt(24);

3. text ellipsis - multi

text ellipsis - multi 역할
순서 역할 코드 예시
1 유동적 폰트 길이 줄임 정의
또는,
ellipsis-multi(전달된 인수에 별도로 지정 가능)		 @mixin ellipsis-multi

_mixins.scss		 

	@mixin ellipsis-multi($line: 2, $font-size: 16, $line-height: $text-line-height-of-16) {
		// 맵에서 해당 글꼴 크기에 맞는 스타일 조회
		$style: if(map-has-key($typography-styles, $font-size), map-get($typography-styles, $font-size), null);
		// 스타일이 맵에 없는 경우 기본값 적용
		@if ($style == null) {
			$style: ($font-size, $text-line-height-of-16); // 기본값 설정
		}
		// 변수 할당
		$font-size: nth($style, 1);
		$line-height: nth($style, 2);
		// line-height 퍼센트 계산
		$line-height-percentage: calc($line-height / $font-size);
		//css 스타일 영역
		overflow: hidden;
		text-overflow: ellipsis;
		display: -webkit-box;
		-webkit-box-orient: vertical;
		-webkit-line-clamp: $line;
		word-wrap: break-word;
		line-height: $line-height;
		height: calc(calc($line-height-percentage * $font-size) * $line); // px 단위 추가
		font-size: $font-size;
	}

✅ 실 사용 예시


		@include ellipsis-multi($line: 2, $font-size: 14px);

4. opacity

opacity 역할
순서 역할 코드 예시
1 투명도와 컬러 함수 정의 _to_opacity(컬러값, hex코드)

_mixins.scss		 

	@function __to_opacity($color, $hex) {
		@return #{$color + $hex};
	}

✅ 실 사용 예시


	background-color: __to_opacity($color-white,'CC');
	box-shadow: 0px 10px 20px __to_opacity($color-black, '0D');
