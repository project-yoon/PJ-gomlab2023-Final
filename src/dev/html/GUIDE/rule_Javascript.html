Basic Javascript Rule

Javascript 작성 규칙

  • 클래스명은 스네이크 표기법(snake_case) 또는, 카멜 표기법(camelCase)을 따른다.
  • 예약어또는, 위치와 기능의 클래스명은 스네이크 표기법(snake_case)으로 작성한다.
  • 이외, 위치 지정 없이 한 단어 이상의 띄어쓰기를 구분할 때, 카멜 표기법(camelCase)으로 작성한다.

	function openPopup(el) {
		if (!el.dataset) {
			return;
		}
	
		const _dataSet = el.dataset.role.replace('open__', '');
		const _targetName = `${_dataSet}.html`;
	
		el.addEventListener('click', e => {
			e.preventDefault();
	
			console.log(e.target)
	
			let buttonDim = e.currentTarget.dataset.closeLayer === 'false'
	
			fetch(_targetName).then(res => {
				if (!res.ok){
					throw new Error;
				}
				return res.text()
			}).then(res => {
				makeLayer(res, buttonDim);
			}).catch(err => {
				console.log(err);
			})
		});
	};
