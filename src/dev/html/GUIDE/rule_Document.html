기본정책

기본정책
인코딩 UTF-8
마크업 종류 HTML5, gulp-sass, Javascript
웹 유형 PC Web , Mobile Web
PC 웹 호환성 chrome (버전 120.0), Edge 버전 120.0 이상
Mobile 웹 호환성
device : Android 5.0 이상, IOS 15.6 이상

browser : 사파리, 크롬
해상도 정책 　- PC UI : 최대 1920px

　- MO UI : 375px ~ (as-is:320)

　- tab : 767px ~ 1279px, (asis기준)

Gulp plugin list

  • gulp-html-tag-include
  • browser-sync
  • gulp-sourcemaps
  • gulp-sass
  • node-sass(gulp-sass버전이4.x.x대라서 사용)

※ 해당 리스트는 제작 초기 세팅이며, 제작 완료 또는 운영중에 변경이 가능한 부분이기에 필수 요건에 해당하지 않습니다.

META

- Encoding


	<meta charset="UTF-8">

- 브라우저 호환성 설정


	<meta http-equiv="X-UA-Compatible" content="IE=Edge">

- 모바일 설정


	<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0, user-scalable=no">

Document full format.

(참고) <TEMP_BASIC_PAGE.html>


	<!DOCTYPE html>
	<html lang="ko">
		<include src="../include/head.html" title="페이지 타이틀"></include>
		<body> ← ★ 페이지 키클래스
			<!-- 헤더 -->
			<include src="../include/header.html"></include>
			<!-- //헤더 -->
		
			<!-- 컨텐츠 -->
			<main id="container">
				<div id="contents" class="contents-body">
					<section class="panel-wrap">
						<div class="panel">
							내용입력 : panel-wrap
						</div>
					</section>
				</div>
			</main>
			<!-- //컨텐츠 -->
		
			<!-- 퀵 메뉴 -->
			<include src="../include/quick_menu.html"></include>
			<!-- //퀵 메뉴 -->

			<!— ★ 각 화면 별 단독 js 영역 -->
			<script>
				console.log('여기는 화면 전용 js구현영역 입니다.')
			</script>
			<!— //각 화면 별 단독 js 영역 -->

			<!-- 푸터 -->
			<include src="../include/footer.html"></include>
			<!-- //푸터 -->
		</body>
	</html>
