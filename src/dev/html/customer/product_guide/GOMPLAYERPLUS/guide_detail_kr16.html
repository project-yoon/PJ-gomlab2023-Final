음독 서비스

한국어 자막의 발음을 알파벳, 카타카나로 표기해 나타내는 서비스입니다.

음독 서비스
음독 서비스 해당 버튼 혹은 F6키를 통해 음독 서비스 팝업 창으로 이동합니다.
  • 동영상 열기

    음독 서비스와 함께 시청할 동영상을 불러옵니다.

    곰플레이어 플러스의 빈 화면을 우클릭한 후 [열기] > [주소 열기]를 클릭하거나, 단축키 [Ctrl+U]를 누르면 다음과 같은 창이 나타납니다.

    상단 주소 입력 란에 동영상의 웹 주소를 직접 입력하거나, 주소 파일 목록에서 원하는 M/V를 선택해 재생할 수 있습니다

  • 음독 기능 설정
    음독 기능 설정
    음독 서비스 해당 버튼을 클릭해 음독 기능을 ON/OFF 할 수 있습니다.
    표시 문자 음독 표기 문자를 선택합니다.
    알파벳(영문), 카타카나(일문) 두 가지 옵션이 있습니다.
    [자막 설정] 자막의 폰트, 스타일 등을 변경하는 설정창으로 이동합니다.
    [싱크 가사 찾기] M/V에 맞는 가사를 검색 및 지정하는 팝업창으로 이동합니다.
  • 싱크 가사 찾기

    해당 팝업 창에서 M/V에 맞는 가사를 검색하고 적용할 수 있습니다.

    싱크 가사 찾기
    M/V에 맞는 가사를 검색합니다.
    음독 기능을 ON/OFF 합니다.
    선택한 가사의 Html파일을 다운로드합니다.
    해당 파일에는 타임스탬프, 한국어 자막, 알파벳 음독이 표기되어 있습니다.
    검색어에 맞는 가사 목록이 표시됩니다.
    선택한 M/V의 가사가 표시됩니다.

    초기에는 재생 중인 M/V 제목에 맞는 가사가 자동으로 적용되며, 해당 파일이 적합하지 않을 경우 검색을 통해 변경할 수 있습니다.

    예를 들어, BTS의 ‘DNA’를 검색창에 입력했을 경우 결과는 다음과 같습니다.

    검색 결과 목록에서 원하는 가사를 더블클릭하면 영상에 적용됩니다.

    현재 상태에서 가사가 나오는 영역을 클릭하면 다음과 같은 화면으로 변환됩니다.

    싱크 가사 찾기
    스크롤을 통해 현재 재생 구간을 벗어났을 경우 버튼이 붉은색으로 활성화되며, 클릭할 시 현재 재생중인 구간으로 스크롤이 이동합니다.
    가사를 탐색합니다. 특정 가사를 클릭하면 M/V가 해당 구간부터 재생됩니다.
