영상 재샣하기

일반 영상 재생하기

곰플레이어+를 이용하면 PC나 인터넷에 있는 영상을 찾아 손쉽게 감상할 수 있습니다.

기본 메뉴의 열기 메뉴를 선택하면 나타나는 팝업 메뉴에서 PC 또는 인터넷에 있는 영상을 불러올 수 있습니다.

간편 열기: 간편 열기 창을 이용하여 PC에 있는 영상 파일을 불러옵니다.

파일 열기: 윈도우 탐색기를 이용하여 PC에 있는 영상 파일을 불러옵니다.

주소 열기: 인터넷에 있는 영상이나 방송에 연결합니다.

폴더 열기: 선택한 폴더 안에 있는 영상 파일을 모두 불러옵니다.

기본 열기 설정: 주 화면 하단에 있는 열기() 아이콘을 클릭하면 나타나는 파일 열기 방법을 설정할 수 있습니다.

PC 에 있는 영상 재생하기

PC에 있는 영상을 재생하는 방법은 다음과 같습니다.

기본 메뉴의 열기 메뉴를 선택하면 나타나는 팝업 메뉴에서 PC 또는 인터넷에 있는 영상을 불러올 수 있습니다.

  • 1. 기본 메뉴에서 열기 > 파일 열기 메뉴를 선택하세요.
  • 2. 재생하고자 하는 파일을 선택한 후, 열기 버튼을 클릭하세요

인터넷 영상 재생하기

인터넷에 있는 영상이나 방송을 재생하는 방법은 다음과 같습니다.

  • 1. 기본 메뉴에서 주소 열기 메뉴를 선택하세요.
  • 2. 주소 입력칸에 영상이나 방송이 있는 인터넷 주소를 직접 입력하세요.
    주소 파일 목록에서 원하는 주소를 선택할 수 있습니다.

    주소 파일 목록에 있는 추가 또는 삭제 버튼을 클릭하여 새로운 주소를 추가하거나 기존에 입력된 주소를 삭제할 수 있습니다
    최근 사용 주소 탭을 클릭하면, 최근에 접속하여 재생했던 주소가 순서대로 표시됩니다.

  • 3. 확인 버튼을 클릭하세요. 해당하는 영상 또는 방송이 자동으로 재생됩니다.
