코덱찾기

곰플레이어+에는 일반적인 영상 파일을 재생할 때 필요한 비디오 코덱이 내장되어 있어 대부분의 영상 파일을 재생할 수 있습니다. 하지만 일부 특수하게 제작된 영상들은 해당 영상에 적합한 코덱을 설치한 후 재생할 수 있습니다. 곰플레이어+는 특수한 코덱이 필요한 경우, 해당 코덱을 쉽게 찾고 설치할 수 있도록 도와줍니다.

외부 코덱의 설치가 필요한 영상 파일을 재생하면 코덱 찾기 창이 나타납니다. 이때 코덱찾기 버튼을 클릭하면 필요한 코덱의 정보나 해당 코덱을 다운로드할 수 있는 경로를 알려줍니다.
