고정 영역 캡처

고정 영역 캡처에는 영역을 지정하고 한 장씩 캡처하는 기본 모드와 캡처 간격과 개수를 설정하여 여러 장 캡처하는 연속 캡처 모드가 있습니다. 고정 영역 캡처는 캡처할 영역을 지정한 후 편리하게 캡처하고 툴바에서 모드를 설정하거나, 그리기 도구로 다양한 효과를 추가할 수 있습니다.

1. 고정 영역 캡처

- 기본 모드

고정 영역 캡처 - 기본 모드
이미지 캡처 이미지 캡처 영역에 표시된 화면을 이미지로 캡처할 수 있습니다. 그리기 도구를 이용하여 추가한 효과들도 함께 캡처 됩니다.
그리기 그리기 도구로 다양한 효과를 추가할 수 있습니다. 그리기 아이콘 클릭 시 그리기 툴바가 화면에 세로로 나타납니다. 툴바를 사용해서 화면에 그림을 그리거나 스탬프, 도형, 텍스트 등을 추가해 보세요.
영역 다시 설정 캡처 영역을 다시 설정할 수 있습니다. 영역 다시 설정 아이콘을 클릭하면 마우스로 녹화하려는 창을 선택하거나 마우스를 드래그하여 녹화 영역을 다시 설정할 수 있습니다.
연속 캡처 설정 아이콘 클릭 시 연속 캡처 설정 팝업이 나타나며 연속 캡처 on/off 할 수 있습니다.
닫기 이미지 캡처 툴바가 사라지고 홈 화면이 나타납니다.

- 연속 캡처 모드

고정 영역 캡처 - 연속 캡처 모드
연속 캡처 클릭 시 설정된 값에 따라 이미지가 연속으로 여러 장 캡처 됩니다. 캡쳐 간격과 개수는 연속 캡처 설정에서 변경할 수 있습니다.
그리기 그리기 도구로 다양한 효과를 추가할 수 있습니다. 그리기 아이콘 클릭 시 그리기 툴바가 화면에 세로로 나타납니다. 툴바를 사용해서 화면에 그림을 그리거나 스탬프, 도형, 텍스트 등을 추가해 보세요.
영역 다시 설정 캡처 영역을 다시 설정할 수 있습니다. 영역 다시 설정 아이콘을 클릭하면 마우스로 녹화하려는 창을 선택하거나 마우스를 드래그하여 녹화 영역을 다시 설정할 수 있습니다.
연속 캡처 설정 아이콘 클릭 시 연속 캡처 설정 팝업이 나타나며 연속 캡처 설정을 on하여 캡처 간격 시간과 개수를 설정할 수 있습니다.
닫기 이미지 캡처 툴바가 사라지고 홈 화면이 나타납니다.

2. 고정 영역 캡처하기

다음 순서대로 이미지 캡처를 진행해 보세요.

  • ① 홈 화면에서 [이미지 캡처] 메뉴 우측 상단에 있는 [더 보기] 버튼을 클릭하고 [고정 영역 캡처] 메뉴를 클릭합니다.
  • ② 마우스를 드래그하여 캡처할 영역을 선택하면 이미지 캡처 툴바가 노출됩니다.
  • ③ [이미지 캡처] 버튼을 클릭하면 캡처가 완료됩니다.
    • 연속 캡처 진행 시 [연속 캡처 설정 ] 버튼을 클릭하여 설정 팝업에 연속 캡처 설정을 on으로 변경하고 연속 캡처 간격과 개수를 설정합니다.
    • 설정 후 [이미지 캡처] 버튼 클릭 시 실시간으로 캡처하는 수를 확인하실 수 있습니다.
  • ④ [닫기] 버튼을 클릭하면 이미지 캡처가 종료되며 홈 화면이 나타납니다. 탐색기 영역에서 캡처한 이미지 파일을 확인할 수 있습니다.
