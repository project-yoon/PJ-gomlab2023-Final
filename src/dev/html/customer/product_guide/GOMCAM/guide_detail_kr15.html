GIF 애니메이션

GIF 애니메이션은 연속된 이미지를 하나의 파일에 담아 애니메이션 이미지로 만드는 기능입니다. GIF 애니메이션 기능을 사용하여 영상의 일부분을 움직이는 이미지 파일을 만들 수 있습니다.

1. 캡처 전 툴바

캡처 전 툴바
캡처 시작 캡처를 시작할 수 있습니다.
영역 다시 설정 캡처 영역을 다시 설정합니다. 영역 다시 설정 아이콘 클릭한 후 마우스를 드래그하여 캡처 영역을 다시 설정할 수 있습니다
GIF 애니메이션 캡처 설정
 클릭 시 GIF 애니메이션 캡처 설정 팝업이 나타나며 FPS와 속도를 변경할 수 있습니다.
닫기 캡처 툴바가 사라지고 홈 화면이 나타납니다.

2. 캡처 중 툴바

캡처 전 툴바
일시 정지
다시 시작 		캡처를 일시 정지할 수 있습니다. 일시 정지 중에는 [일시 정지] 아이콘이 [다시 시작] 아이콘으로 변경되며, 아이콘을 클릭하면 녹화가 다시 시작됩니다.
캡처 종료 캡처가 종료되며 [GIF 미리 보기] 팝업이 나타납니다. 팝업에서 캡처한 GIF 애니메이션을 확인하고 편집할 수 있습니다.
캡처 시간 실시간으로 캡처된 시간을 확인할 수 있습니다.
닫기 캡처 툴바가 사라지고 홈 화면이 나타납니다. 만약 캡처 중 아이콘을 클릭했다면 [GIF 미리 보기] 팝업이 나타나 캡처한 GIF 애니메이션을 확인하고 편집할 수 있습니다.

3. GIF 애니메이션 캡처하기

다음 순서대로 GIF 애니메이션 캡처를 진행해보세요.

  • ① 홈 화면에서 [ 이미지 캡처] 버튼의 우측 상단에 있는 [더 보기] 버튼을 클릭하고 [GIF 애니메이션] 메뉴를 클릭합니다.
  • ② 캡처 툴바가 나타납니다. 툴바를 통해 캡쳐 시작, 영역 다시 설정, 캡처 설정을 할 수 있습니다.
  • ③ [캡처 시작 ] 버튼을 클릭해 GIF 애니메이션 캡처를 시작합니다.

    [GIF 애니메이션 캡처 설정 ] 버튼을 클릭해 FPS와 속도를 설정할 수 있습니다.

  • ④ 캡처가 끝나면 [캡처 종료] 버튼을 클릭합니다. 캡처가 종료되며 [GIF 미리 보기] 팝업이 나타나 저장된 GIF 애니메이션 파일을 확인할 수 있습니다.
  • ⑤ [GIF 미리 보기] 팝업에서 [편집] 버튼을 클릭하면 [GIF 편집하기] 팝업이 나타납니다. 편집 팝업을 통해 저장된 GIF 애니메이션 파일을 간단히 편집할 수 있습니다. 편집 후 내보내기 시 GIF 애니메이션 파일로 저장됩니다.
  • ⑥ [닫기] 버튼을 클릭하면 캡처가 종료되며 홈 화면이 나타납니다. 탐색기 영역에서 GIF 애니메이션 파일을 확인할 수 있습니다.
