자동 스크롤 캡처

자동 스크롤 캡처는 웹 브라우저 화면에 스크롤 바가 있을 경우, 브라우저를 자동으로 스크롤하면서 전체 내용을 하나의 이미지로 캡처하는 기능입니다.

다음 순서대로 GIF 애니메이션 캡처를 진행해보세요.

  • ① 홈 화면에서 [이미지 캡처] 메뉴 우측 상단에 있는 [더 보기] 버튼을 클릭하고 [자동 스크롤 캡처] 메뉴를 클릭합니다.
  • ② 캡처할 웹페이지에 마우스를 올려놓으면 스크롤 방향을 선택하는 아이콘이 나타납니다.
    (스크롤 방향 아이콘이 표시되지 않을 경우, 현재 화면에 보이는 화면만 캡처 됩니다.)
  • ③ 원하는 스크롤 방향 아이콘을 클릭합니다.
    • : 지그재그로 스크롤하며 캡처합니다.
    • : 위에서 아래 방향으로 스크롤하며 캡처합니다.
    • : 왼쪽에서 오른쪽 방향으로 스크롤하며 캡처합니다.

    * 캡처 중에는 마우스 사용이 제한되며 클릭 시 캡처가 중지됩니다.

  • ④ 캡처가 완료되면 캡처가 종료되며 홈 화면이 나타납니다. 탐색기 영역에 저장된 이미지 파일을 바로 확인할 수 있습니다.
