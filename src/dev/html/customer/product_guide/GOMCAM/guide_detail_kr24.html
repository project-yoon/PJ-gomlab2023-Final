편집 도구

1. 그리기 도구

상단에 위치한 [그리기] 도구 클릭 시 하단에 그리기 기능을 제공하는 다양한 툴과 옵션을 확인할 수 있습니다.
각 도구 클릭 시 선택된 툴의 색상이나 크기, 두께 등을 조절할 수 있는 옵션 팝업이 나타납니다.

그리기 도구
펜 펜으로 선을 그릴 수 있습니다. [펜] 버튼을 클릭하면 설정 팝업이 나타나고 색상 및 두께를 설정할 수 있습니다.
형광펜 형광펜으로 선을 그릴 수 있습니다. [형광펜] 버튼을 클릭하면 설정 팝업이 나타나고 색상 및 두께를 설정할 수 있습니다.
도형 기본 도형인 사각형, 원, 선, 화살표를 추가할 수 있습니다. [도형] 버튼을 클릭하면 설정 팝업이 나타나고 도형의 종류, 색상 및 크기를 설정할 수 있습니다.
스탬프 이모티콘, 별, 체크 등 스탬프를 추가할 수 있습니다. [스탬프] 버튼을 클릭하면 설정 팝업이 나타나고 스탬프의 종류, 색상 및 크기를 설정할 수 있습니다.
텍스트 텍스트를 추가할 수 있습니다. [텍스트] 버튼을 클릭 후 녹화 영역 원하는 위치에 클릭 시 텍스트를 작성해 보세요. 텍스트 설정 팝업에서 텍스트의 색상과 크기를 설정할 수 있습니다.
지우개 지우개 아이콘 클릭 시 두께 설정 팝업이 나타나며, 두께 조절이 가능합니다. 클릭, 드래그하여 원하는 영역을 지울 수 있습니다.
모두 삭제 모두 삭제 아이콘 클릭 시 이미지 안에 추가한 효과들이 전부 삭제됩니다.

2. 자르기 도구

상단에 위치한 [자르기] 도구 클릭 시 하단에 자르기, 대칭, 회전 기능을 확인할 수 있습니다.

자르기 도구
영역 지정 사용자지정, 원본, 정사각형을 각 선택하여 비율을 고정하거나 비율고정 없이 영역 지정을 할 수 있습니다.
영역 크기 현재 영역의 크기 정보가 나타나며 수동 입력으로 영역 크기는 변경 가능합니다. 또한 체인 활성화 시 비율을 유지하여 영역 크기 변경이 가능합니다.
재설정 [재설정] 버튼 클릭 시 현재 지정된 영역이 취소되며 맨 처음 자르기 전 상태로 복구됩니다.
적용 [적용] 버튼 클릭 시 선택된 영역을 제외한 모든 영역을 잘라냅니다.
회전 좌, 우 90도씩 선택 영역과 상관없이 이미지 전체가 회전됩니다.
대칭 상/하, 좌/우로 선택 영역과 상관없이 이미지 전체가 대칭됩니다.

3. 이미지 크기 변경 도구

상단 좌측에 위치한 자르기 [도구] 클릭 시 창 중간에 이미지 크기 조정 팝업이 나타나며 이미지 크기를 조정을 할 수 있습니다.

  • 픽셀과 백분율 두 가지 중 선택할 수 있으며 값을 사용자가 입력하여 크기 조정이 가능합니다.
  • [비율 유지] 아이콘 클릭 시 비율을 유지하여 이미지 크기 변경 가능합니다.
  • [적용] 버튼 클릭 시 조정된 이미지 크기로 변경되어 나타나며 [취소] 버튼 클릭 시 크기 조정 창이 닫힙니다.
