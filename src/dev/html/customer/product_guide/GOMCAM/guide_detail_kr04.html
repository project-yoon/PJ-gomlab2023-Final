녹화 및 캡처 메뉴

1. 화면 녹화

[ 화면 녹화] 버튼을 클릭 시 PC 화면에서 녹화할 영역을 지정하면 녹화 툴바가 노출됩니다. 녹화 툴바를 이용하여 화면을 녹화하고 녹화 시 사용할 수 있는 부가 기능을 이용해보세요.

[더 보기] 버튼을 클릭하면, 더 다양한 녹화 기능을 이용할 수 있습니다.
가장 최근에 이용한 녹화 기능은 ‘화면 녹화’ 버튼 아래에 새로운 버튼으로 추가됩니다.
예를 들면 [더 보기]버튼 후 예약 녹화 클릭 시 [예약 녹화] 버튼이 [화면 녹화] 버튼 아래 첫 번째 순서로 나타나며 다음 녹화 시 빠르게 해당 녹화 기능을 이용할 수 있습니다.

화면 녹화 - 더보기
전체 화면 녹화 PC 화면 전체를 녹화할 수 있습니다. 2개 이상의 모니터가 연결된 경우 녹화를 원하는 1개의 모니터를 선택해야 합니다.
웹캠 녹화 PC에 연결된 웹캠 장치를 선택한 후 녹화할 수 있습니다.
강의 녹화 녹화할 PowerPoint, HanShow, PDF 문서를 선택한 후 슬라이드 쇼(전체 화면) 모드로 녹화할 수 있습니다.
예약 녹화 예약 녹화는 특정 일자와 시간에 녹화를 시작하여 정해진 시간만큼 자동으로 녹화하는 기능입니다. 예약 녹화 팝업에서 원하는 일정의 녹화 예약을 추가하거나 수정할 수 있습니다.
직접 입력 녹화 영역의 크기를 직접 설정해 녹화할 수 있습니다.
최근 녹화 영역 최근 녹화한 영역을 다시 녹화할 수 있습니다.

2. 이미지 캡처

[ 이미지 캡처] 버튼을 클릭한 후 PC 화면에서 이미지 캡처할 영역을 지정하면 캡처가 완료됩니다.

[더 보기] 버튼을 클릭하면, 더 다양한 캡처 기능을 이용할 수 있습니다.
가장 최근에 이용한 캡처 기능은 [이미지 캡처] 버튼 아래에 새로운 버튼으로 추가되어 빠르게 이용할 수 있습니다.

이미지 캡처 - 더보기
전체 화면 캡처 PC화면 전체를 캡처할 수 있습니다. 2개 이상의 모니터가 연결된 경우 캡처를 원하는 1개의 모니터를 선택해야 합니다.
고정 영역 캡처 캡처 툴바를 이용하여 원하는 영역을 연속으로 캡처할 수 있습니다.
다중 모니터 캡처 PC에 연결된 모든 모니터의 화면을 한 번에 캡처할 수 있습니다.
GIF 애니메이션 GIF 애니메이션은 연속된 이미지를 하나의 파일에 담아 애니메이션과 같이 만드는 것입니다. 캡처 툴바를 이용하여 원하는 GIF 애니메이션을 만들 수 있습니다.
자동 스크롤 캡처 웹 브라우저 화면에 스크롤 바가 있을 경우, 브라우저를 자동으로 스크롤하면서 전체 내용을 하나의 이미지로 캡처할 수 있습니다.
최근 캡처 영역 최근 캡처한 영역을 다시 캡처할 수 있습니다.
