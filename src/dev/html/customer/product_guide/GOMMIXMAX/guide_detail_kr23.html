타임라인 패널

타임라인 작업 영역

클립들을 이동하거나 편집하는 작업 영역입니다.

1. 클립 선택

타임라인 작업 영역으로 옮겨진 클립을 마우스로 클릭하면 클립이 선택됩니다. 선택된 클립은 프레임이 생깁니다. 텍스트/오브젝트 클립을 마우스로 더블 클릭하면 [속성 편집] 창이, 이외의 클립을 마우스로 더블 클릭하면 [이펙트 속성] 창이 나타납니다. 미리보기 패널 모니터 화면은 [트랙 클립 모드]로 전환됩니다. 이때 타임라인 재생 헤드는 클립 맨 앞으로 이동되며 비활성화 됩니다. 이펙트 속성 지정 상태에서 재생 헤드의 이동은 미리보기 패널의 재생 헤드 또는 키 프레임 컨트롤 재생 헤드를 사용하십시오.

2. 클립 이동

선택된 클립을 마우스 왼쪽 버튼을 누른 상태에서 마우스를 드래그하면 클립이 이동됩니다. 클립은 들어갈 공간이 확보되어 있으면 붉은색 테두리가 보이며 마우스를 드랍 시 해당 위치로 이동됩니다. 공간이 없다면 이동되거나 삽입되지 않습니다. 트랙 간의 이동이 되지만 오디오 정보가 없는 클립,텍스트, 이미지, 선택한 클립을 인접한 클립 앞뒤 또는 공간이 확보되지 않은 위치로 이동 또는 복사하려면 먼저 이동하려는 클립을 선택합니다. 또는 복사를 합니다. 그리고 재생 헤드를 이동시킬 위치로 옮긴 후 붙여넣기를 합니다.

3. 클립 잘라내기/복사/붙여넣기

타임라인 작업 영역에서 선택된 클립은 타임라인 툴바 또는 오른쪽 우클릭 메뉴를 통해 잘라내기/복사/붙여넣기 할 수 있습니다. 잘라내기나 복사를 한 클립은 붙여넣기로 이동/복사됩니다. 붙여넣기는 뒤에 있는 클립을 밀면서 복제하는 방식과 덮어쓰며 복제하는 방식이 있습니다. 선택된 트랙의 재생헤드 위치에 복사됩니다. 재생 헤드가 클립 중간에 위치하면 클립을 자른 후 삽입되거나 덮어씁니다. 오른쪽 우클릭 메뉴의 “현재 마우스 위치에 복제”를 하면 마우스가 놓여진 위치에 복사됩니다.

4. 클립 길이 조절

비디오/오디오 클립은 트림이 적용되는 방식으로 길이를 조정합니다. 마우스 포인터를 클립의 맨 앞이나 맨 뒤쪽으로 옮기면 클립의 앞/뒤 선택 부분이 활성화됩니다. 마우스를 눌러서 좌우로 움직이면 클립의 길이가 조정됩니다. 타임라인 툴 바의 오른쪽 영역에 보이는 클립 길이 정보를 참고하여 조정합니다. 미세한 길이 조정을 위해 타임라인의 보기 배율을 충분히 확대한 후 조정하십시오.

[배속] 설정창에서 클립 길이를 변경할 수 있습니다. 배속을 빠르게 적용하면 클립의 길이가 줄어들고, 배속을 느리게 적용하면 클립의 길이가 늘어납니다.

배속 설정창
배속 비디오 클립의 재생 배속입니다. 배속을 변경하면 배속 값에 따라 재생 시간이 자동으로 변경됩니다.
재생 시간 클립의 재생 시간입니다. 재생 시간을 변경하면 배속 값이 자동으로 변경됩니다.

클립을 2배 느리게 재생하려면 배속을 50%로 변경합니다. 속도가 느려져서 재생 시간은 2배로 길어집니다. 2배 빠르게 재생하려면 배속을 200%로 변경합니다. 속도가 빨라져서 재생 시간은 2배로 짧아집니다. 반대로 재생 시간을 변경하여 배속을 변경할 수 있습니다. 배속 조정으로 클립이 늘어나면 뒤 클립을 덮어쓰지 않고 초과되는 길이는 트림 처리됩니다.

5. 클립 음소거

클립 선택 후 오른쪽 우클릭 > 음소거 선택 시 클립 하단에 가 표시됩니다. 다시 한 번 음소거 선택 시 음소거가 해제됩니다.

6. 이펙트 속성

클립에 적용되어 있는 이펙트 리스트를 보여줍니다. 이펙트 별로 속성값을 변경, 삭제하거나 순서를 바꿉니다.

이펙트 속성
이펙트를 선택합니다. 선택되면 해당 이펙트 속성창이 펼쳐집니다.
이펙트의 순서를 바꿉니다. 맨 위로/위로/아래로/맨 아래로
선택된 이펙트의 속성값을 초기화합니다.
선택된 이펙트를 삭제합니다.
