프로젝트 설정

새 프로젝트 파일의 동영상 해상도와 1초당 만드는 프레임 수를 설정합니다. 새 프로젝트에서 클립을 타임라인에 넣기전에 설정 변경해야 합니다. 설정 값은 바로 전에 사용한 프로젝트 파일의 설정값을 유지합니다. 타임라인에 클립이 추가하면 이 메뉴는 비활성화 됩니다.

프로젝트 설정
종횡비 동영상의 가로/세로 비율을 지정합니다. 일반적으로 가로형 동영상은 16:9의 비율이며, 세로형 동영상은 9:16의 비율입니다.
특정 플랫폼에서 사용되는 비율로 사용하려면 종횡비를 선택하고 해상도를 선택합니다. 선택할 종횡비가 없다면 사용자 설정을 선택한 후 해상도를 직접 입력합니다.
해상도 선택된 종횡비의 동영상 해상도 목록을 보여주며 여기에서 동영상 해상도를 선택합니다. 원하는 해상도가 없다면 “사용자 설정”을 선택한 후 해상도를 직접 입력합니다.
프레임/초 동영상의 1초당 만들어지는 프레임 수를 선택합니다.
최대 60프레임까지 지정할 수 있으며 기본값은 1초당 25프레임입니다.
