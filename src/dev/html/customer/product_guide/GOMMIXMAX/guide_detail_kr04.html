빠른 학습

간단한 동영상 제작

  • 1. 새 프로젝트 파일을 만듭니다. 홈 화면에서 새 프로젝트를 선택합니다.
  • 2. '프로젝트 설정'에서 동영상 해상도, 프레임 수를 설정합니다. 한번 설정된 값은 계속 유지됩니다. 프로젝트 설정을 하고 프로젝트 파일을 저장한 후 작업하십시오.
  • 3. 편집하려는 클립들을 프로젝트 클립 리스트로 불러옵니다. 미디어 클립을 불러옵니다. 텍스트 클립을 추가합니다. 필요시 백그라운드 /오브젝트 클립을 만듭니다.
  • 4. 프로젝트 클립 리스트에 등록된 클립을 타임라인 패널에 넣습니다. 타임라인 패널의 도구 바 중 하나를 선택하여 타임라인 트랙에 넣거나 클립을 타임라인 패널 원하는 트랙에 드래그 앤 드롭(drag & drop)하세요.
  • 5. 타임라인 작업 영역에 담긴 클립을 배치하고 편집합니다. 클립을 자르거나 이동하고 클립에 이펙트를 적용합니다.
  • 6. '내보내기'를 실행하여 새로운 동영상을 만듭니다. 완료 후 새로 만들어진 동영상을 확인합니다.
