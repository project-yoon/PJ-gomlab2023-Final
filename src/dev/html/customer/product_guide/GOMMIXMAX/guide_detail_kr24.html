환경 설정

프로그램 사용 및 동영상 편집에 관련된 여러가지 환경을 설정합니다. 프로그램 언어 설정, 프로젝트 파일의 저장 폴더 위치나 인코딩되는 출력 폴더 위치, 이름 지정 형식 및 새로 생성되는 클립의 길이 등을 지정합니다.

파일 메뉴에서 '환경 설정'으로 들어 가면 환경 설정 대화상자가 나타납니다.

1. 일반

클립들을 이동하거나 편집하는 작업 영역입니다.

일반
언어설정 프로그램의 언어를 설정합니다.
프로젝트 경로 설정 프로젝트 파일이 저장되는 폴더 위치입니다. 버튼을 누른 후 폴더를 변경합니다.
프로젝트 이름 설정 새 프로젝트 파일을 만들 때 임시로 적용되는 프로젝트 파일 이름입니다.
임시로 적용된 프로젝트 파일 이름이므로 최초 저장할 때 저장 대화상자가 나타납니다.
초기값 레이아웃으로
새 프로젝트 시작		 체크되면 새 프로젝트를 만들 때 프로젝트 패널과 소스 및 효과 패널, 미리보기 패널, 타임라인 패널의 레이아웃 비율을 초기 비율로 설정해서 새 프로젝트를 시작합니다.
트랙 추가 시 트랙 트랜지션 확장 트랙 추가 시 트랙 트랜지션 확장을 결정할 수 있습니다.
자동 종료 설정 종료 설정 없음 / 프로그램 종료 / 시스템 종료 중에 선택할 수 있으며 '종료 설정 없음'이 기본적으로 설정되어 있습니다.

2. 편집

편집
재생 시간(초) 비디오나 오디오 클립은 재생 시간을 갖고 있습니다. 그러나 텍스트/이미지/백그라운드/오브젝트 클립 및 클립, 트랙 트랜지션은 사용자가 재생 시간을 지정합니다. 해당 클립을 만들면 여기에 설정된 시간으로 클립을 만듭니다. 클립 기본 값은 4초이며 클립, 트랙 트랜지션은 2초입니다. 이 값을 10초로 변경하면 클립 및 효과는 10초 길이로 만듭니다. 클립의 길이는 타임라인 작업 영역에서도 변경할 수 있습니다.

3. 저장

저장
내보내기 경로 설정 “내보내기” 할 때 인코딩되는 동영상을 저장하는 폴더 경로입니다. 버튼을 누른 후 폴더를 변경합니다.
- 내보내기 이름 설정
내보내기 이름 설정
프리픽스 사용 체크되면 인코딩되는 파일명 앞쪽에 지정된 스트링을 포함하여 저장합니다.
해상도 체크되면 인코딩되는 파일명 뒤쪽에 해상도를 포함하여 저장합니다.
날짜와 시간 체크되면 인코딩되는 파일명 뒤쪽에 날짜와 시간을 포함하여 저장합니다.

예를 들면 ClipRoaster_1280x720_2021-03-01.mp4 이런 유형으로 저장됩니다.
기본값은 해제가 되어 있기에 “프로젝트명.mp4”로 저장됩니다.

스냅샷
포맷 BMP/JPG/GIF/PNG 형식 중 스냅샷을 저장합니다.
경로 설정 스냅샷 파일이 저장되는 폴더 위치입니다.
파일 이름 스냅샷 파일이 저장되는 파일 이름입니다. [클립 파일로 추가]를 설정할 수 있습니다.
자동 백업 설정 일정 시간마다 자동으로 저장될 수 있도록 설정합니다. 기본 값은 4분이며,
화살표 버튼을 클릭하거나 숫자를 직접 입력하여 원하는 시간 간격을 설정할 수 있습니다.
