빠른 학습

단축키

1. 일반

일반 단축키
프로젝트 이름 Ctrl + N 복사 Ctrl + C
프로젝트 열기 Ctrl + O 붙여넣기 (덮어쓰기) Ctrl + V
미디어 클립 불러오기 Ctrl + I 붙여쓰기(삽입) Ctrl + Shift + V
저장 Ctrl + S 복제 Ctrl + D
다른 이름으로 저장 Ctrl + Shift + S 삭제 Delete
프로젝트 전체 저장 Ctrl + Shift + E 내보내기 Ctrl + E
실행취소 Ctrl + Z 프로그램 종료 Alt + F4
다시 실행 Ctrl + Y 프로그램 정보 F1
잘라내기 Ctrl + X 환경 설정 F5

2. 타임라인

타임라인 단축키
클립 추가 (타임라인) Q 마그네틱 ON/OFF Alt + P
선택된 클립 뒤에 추가(삽입) W 재생 헤드로 클립 밀기 Ctrl + Right(→)
자른 후 클립 추가(삽입) E 재생 헤드로 클립 당기기 Ctrl + Left(←)
자른 후 클립 추가(덮어쓰기) R 비디오 트랙 추가 Shift + V
클립 당기며 삭제 Ctrl+Delete 오디오 트랙 추가 Shift + A
클립 자르기 Ctrl + K 트랙 트랜지션 확장 ON/OFF Ctrl + Q
배속/길이 변경 Alt + R 트랙 트랜지션 전체 확장 ON/OFF Ctrl + Shift + Q
오디오 조정 Alt + D 트랙 확장 ON/OFF M
이펙트/오버레이 속성 Alt + S 트랙 전체 확장 ON/OFF Ctrl + Shift + M
속성 편집 (텍스트/오브젝트) Alt + P 마커 추가/편집 Shift + M
트랙 트랜지션 속성 Alt + T 이전 마커로 이동 Shift + Left(←)
클립 트랜지션 속성 Alt + C 다음 마커로 이동 Shift + Right(→)

3. 미리보기

미리보기 단축키
재생/일시 정지 Space 이전 클립 Alt + Left(←)
정지 Ctrl + Space 다음 클립 Alt + Right(→)
이전 프레임 Left(←) 맨 처음 Home
다음 프레임 Right(→) 맨 마지막 End
