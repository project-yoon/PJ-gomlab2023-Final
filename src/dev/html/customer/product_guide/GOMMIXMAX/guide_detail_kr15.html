오브젝트 탭

영상 제작에 유용하게 사용되는 객체를 독립적인 클립으로 제공합니다.

원하는 클립을 선택합니다. 클립 추가 버튼을 누른 후 속성 편집에서 속성을 설정하고 [확인] 버튼을 누르면 속성이 적용됩니다.
예를 들어 '자막 배경 01' 객체 클립을 타임라인에 끌어다 놓은 후 속성 편집을 진행하면 다음과 같은 설정이 가능합니다.

속성 편집에서 색상, 크기, 위치 등의 변수값을 지정하고 [확인] 버튼을 누르면 적용이 완료됩니다.
