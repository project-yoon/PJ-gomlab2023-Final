내보내기

1. 영상 내보내기

타임라인에서 작업된 클립들을 내보내기를 통해 동영상으로 만듭니다.
파일 메뉴에서 '내보내기'를 하면 내보내기 설정창이 나타납니다.

내보내기 설정창은 품질 항목의 동영상의 포맷, 이름 및 해상도 등을 설정합니다. 해상도에 따른 프리셋(1080p, 720p, 480p)뿐만 아니라 유튜브, 모바일, 인스타그램 프리셋 또한 제공합니다. 설정값을 프리셋에서 선택할 수 있고 사용자가 정한 설정값으로 내 프리셋을 추가할 수 있습니다. 하단의 프리셋 저장 버튼을 누르면 현재 설정된 데이터를 보여주는 대화 상자가 나타납니다. 이름을 입력한 후 [확인] 버튼을 누르면 내보내기 대화상자에 등록됩니다. 프리셋 저장에서는 비디오 해상도는 포함되지 않습니다. 프로젝트 디폴트에는 프로젝트를 시작할 때 최초로 설정한 값이 노출됩니다.

내보내기 설정창
포맷 인코딩할 동영상의 포맷을 지정합니다. 지원하는 포맷은 일반적으로 사용되는 H.264(MPEG-4) 형식인 MP4 포맷과 플래시 비디오 파일(Flash Video File) 포맷인 FLV입니다.
저장 폴더 인코딩될 동영상의 저장되는 폴더 위치입니다. 기본 폴더는 환경 설정에서 설정된 경로를 가져옵니다.
파일 이름 인코딩될 동영상의 이름입니다. 기본 파일 이름 형식은 “프로젝트 파일명_내보내기 설정값”으로 정해집니다. [내보내기 설정값]은 [환경 설정]에서 변경할 수 있습니다.
해상도 동영상의 가로, 세로 구성 픽셀 수를 의미합니다.
재생 시간 동영상의 가로, 세로 구성 픽셀 수를 의미합니다.
프레임/초 1초당 구성되는 화면 프레임 수입니다. 25 fps는 1초당 25개의 화면 프레임으로 만드는 영상입니다.
품질 품질을 제어 한다는 의미는 화면에 설정되는 비트레이트 용량을 제어 한다는 의미입니다. 보통 비트레이트는 크게 할수록 품질은 좋아지나 인코딩/디코딩 시 메모리와 저장 용량을 많이 차지하는 단점이 있습니다. 매우 좋음/좋음/보통 세 가지로 구성되어 있습니다.

해상도 프레임 수는 최초 [프로젝트설정] 시 설정한 값을 가져오며, [고급] 버튼을 클릭하면 상세한 설정을 변경할 수 있습니다.

내보내기가 완료되면 '재생' 버튼이 나타납니다.

내보내기 설정창
미리 보기 내보내기하는 영상을 화면에 보며 인코딩합니다.
폴더 열기 인코딩될 동영상의 저장되는 폴더를 엽니다.
재생 인코딩된 동영상을 기본 설정된 동영상 플레이어로 재생합니다.
닫기 인코딩 대화상자를 닫습니다.

2. 오디오 내보내기

편집한 영상 또는 오디오 클립을 오디오 파일로 만들 수 있습니다. 내보내기 시 프리셋 메뉴에서 [오디오]를 클릭하세요. [오디오]프리셋의 포맷은 MP3로 설정됩니다. 오디오 품질은 매우 좋음[256kbps], 좋음[192kbps], 보통[128kbps]으로 선택할 수 있습니다. 설정 확인 후 [시작]을 클릭하면 오디오 파일로 내보내기 됩니다. 상세 설정은 [고급] 에서 조정할 수 있습니다.

3. 부분 내보내기

1) 선택 영역 내보내기

타임라인의 일부 영역만을 지정해서 내보내기 할 수 있습니다.

  • 영역 선택하기

    영역 선택을 위해 영역 선택바를 노출하는 방법은 두 가지입니다.

    • 방법 1. 타임라인 바로 위 메뉴 바에서 혹은 버튼을 클릭합니다.
    • 방법 2. 트랙을 우클릭한 후 ‘시작점 표시’ 혹은 ‘끝점 표시’를 클릭합니다.
    영역 선택하기
    혹은 ‘시작점 표시’ 타임라인 재생헤드 기준으로 시작점 바가 나타나고 동시에 전체 클립 맨 끝부분에 끝점 바가 나타납니다.
    혹은 ‘시작점 표시’ 타임라인 재생헤드 기준으로 시작점 바가 나타나고 동시에 전체 클립 맨 끝부분에 끝점 바가 나타납니다.
  • 영역 조절하기

    선택 영역을 조절하는 방법은 두 가지입니다.

    • 방법 1. 타임라인에 표시된 녹색의 선택 영역 바를 드래그로 조절합니다.
    • 방법 2. 미리보기 패널에 표시된 녹색의 선택 영역 바를 드래그로 조절합니다.

      스페이스 바, 혹은 재생 버튼을 클릭 시 선택 영역만을 재생합니다.

  • 선택 영역 해제하기

    선택 영역을 해제하는 방법은 두 가지입니다.

    • 방법 1. 트랙을 우클릭 한 후 ‘선택 영역 표시 취소’를 클릭합니다.
    • 방법 2. 타임라인이 선택된 상태에서 Esc키를 누릅니다.
  • 선택 영역 내보내기

    선택 영역을 내보내는 방법은 기존 ‘영상 내보내기’와 동일합니다. 원하는 트랙에 영역 지정을 완료한 후 타임라인 메뉴바 [내보내기] 버튼을 클릭하면 내보내기 설정창이 나타납니다.

2) 선택한 클립 내보내기

타임라인에 생성 후 편집한 일부 클립만을 선택해서 내보내기 할 수 있습니다.

  • 선택한 클립 내보내기

    특정 클립만을 내보내는 방법은 두 가지입니다.

    • 방법 1. 해당 클립을 우클릭 한 후 [선택한 클립 내보내기]를 누릅니다.
    • 방법 2. 해당 클립을 선택한 상태에서 [파일] > [선택한 클립 내보내기]를 누릅니다.
