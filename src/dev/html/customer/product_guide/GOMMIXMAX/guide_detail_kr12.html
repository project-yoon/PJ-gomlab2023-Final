탑바 메뉴 패널

1. 홈

홈
홈 곰믹스 맥스 홈 화면으로 이동합니다.

2. 파일

파일
새 프로젝트 새로운 프로젝트 파일을 만듭니다.
위치는 환경 설정의 프로젝트 경로 설정에서 설정된 폴더에 만듭니다. 기본 폴더 위치는
~문서/GOMMixMax 이며 확장명은 .gmxp (GOMMixMax Project file)입니다.
프로젝트 열기 저장된 프로젝트 파일(*.gmxp)을 불러옵니다.
최근 프로젝트 열기 최근 작업했던 프로젝트 파일을 빠르게 불러옵니다.
저장 프로젝트 파일을 편집한 후 저장합니다. 최초 저장할 때 저장 대화상자가 나타납니다.
다른 이름으로 저장 프로젝트 파일을 다른 이름으로 저장합니다.
프로젝트 전체 저장 경로가 다른 클립들을 지정된 폴더로 복사하여 저장합니다.
프로젝트 설정 새 프로젝트 파일 관련된 해상도, 프레임 수를 설정합니다.
환경 설정 프로그램과 관련된 언어 설정, 프로젝트 및 저장 파일 경로, 편집 환경, 단축키를 설정합니다.
내보내기 편집한 프로젝트 파일을 동영상으로 내보내기(인코딩)합니다.
스냅샷 저장 재생 헤드가 위치한 클립의 화면을 이미지 파일로 내보내기 합니다.
선택한 클립 내보내기 트랙 내에 원하는 클립을 선택해서 내보내기 합니다.
내보내기 폴더 열기 내보내기한 영상이 저장된 폴더를 엽니다.
프로그램 종료 프로그램을 종료합니다.

3. 편집

편집
실행 취소 실행했던 동작을 되돌립니다.
다시 실행 되돌렸던 동작을 다시 실행합니다.
잘라내기 프로젝트 클립의 리스트 창에서 선택된 소스 클립, 선택된 텍스트 내용, 타임라인 작업 영역에서 선택된 클립을 내부 저장소로 잘라내기합니다.
복사 클립이나 선택된 내용을 내부 저장소로 복사합니다.
붙여넣기 내부 저장소에 보관된 클립이나 내용을 붙여넣기합니다.
복제 프로젝트 클립 리스트에 만들어진 백그라운드를 복제합니다.
삭제 프로젝트 클립 리스트에 등록된 클립을 삭제하거나 타임라인 작업 영역에 선택된 클립을 삭제합니다.
클립 당기며 삭제 타임라인 작업 영역에 선택된 클립을 공백없이 당기며 삭제합니다.
마커 추가/편집 타임라인에 마커를 추가해 클립의 위치, 시간 등을 표시합니다.

4. 클립

클립
미디어 클립 불러오기 미디어 클립을 프로젝트 클립 리스트로 불러옵니다.
모션 이미지 클립 만들기 이미지 클립에 나타나기/사라지기 동작 및 크기 설정을 한 후 클립을 만듭니다.
배속 타임라인 작업 영역에서 선택된 클립의 배속을 변경합니다.
이펙트/오버레이 속성 타임라인 작업 영역에서 선택된 클립에 적용된 모든 이펙트를 보여줍니다.
모든 이펙트/오버레이 삭제 타임라인 작업 영역에서 선택된 클립에 적용된 이펙트/오버레이를 모두 삭제합니다.
길이 변경 클립의 길이를 변경합니다.
속성 편집 텍스트/오브젝트 클립 속성을 편집합니다.
오디오 설정 타임라인 작업 영역에서 선택된 클립의 오디오 레벨과 음소거를 설정합니다.
클립 트랜지션 속성 타임라인 작업 영역에서 선택된 클립 트랜지션 컨트롤 클립의 속성을 보여줍니다.
트랙 트랜지션 속성 타임라인 작업 영역에서 선택된 트랙 트랜지션 컨트롤 클립의 속성을 보여줍니다.

5. 보기

보기
소스 및 효과 / 미리보기 패널 교대 소스 및 효과 패널과 미리보기 패널의 위치를 서로 교대합니다.
레이아웃 초기화 윈도우 창의 크기와 레이아웃 창을 초기화합니다.

도움말
영상 가이드 곰믹스 맥스 영상 가이드를 보여줍니다.

튜토리얼 가이드 간편 튜토리얼 팝업으로 편집에 필요한 기본 사용법을 보여줍니다.
자주 묻는 질문 자주 묻는 질문과 답변을 GOMLab 페이지로 이동해 보여줍니다.
1:1 문의/제안하기 GOMLab 페이지로 이동해 불편사항, 개선점, 1:1 문의 및 제안을 할 수 있습니다.
로그인 무료 사용자에 한하여 로그인 창을 보여줍니다.
쿠폰 등록 무료 사용자에 한하여 쿠폰 등록 창을 보여줍니다.
제품 등록 정보 유료 사용자에 한하여 정품 등록 정보를 보여줍니다. 정품 등록이 되어있지 않은 경우 정품을
등록하는 대화상자를 보여줍니다.
프로그램 정보 프로그램 관련 정보를 보여줍니다.
