기능 학습

클립

1. 클립의 정의

클립이란 동영상 편집기에서 구성의 기본이 되는 최소 단위의 미디어 요소로 정의합니다.

클립의 정의
비디오 클립 비디오로 사용되는 MP4, AVI, MOV, FLV 포맷 등의 동영상 파일
이미지 클립 사진, 클립 아트 등으로 사용되는 JPG, PNG, BMP, GIF 포맷 등의 이미지 파일
오디오 클립 배경음악, 효과음으로 사용되는 MP3, WAV 형식의 소리로 구성된 파일
텍스트 클립 타이틀, 자막으로 사용되는 텍스트의 속성을 갖는 곰믹스 맥스 내부 텍스트 파일
백그라운드 클립 동영상의 바탕 화면으로 사용하는 곰믹스 맥스 내부 백그라운드 파일
오브젝트 클립 객체를 프리셋 형태로 제공하는 곰믹스 맥스 오브젝트 파일

비디오 클립, 이미지 클립, 오디오 클립을 일반적으로 미디어(동영상) 클립이라고 표현합니다.
동영상을 제작하려면 타임라인에 최소 1개 이상의 클립이 담겨야 클립을 동영상으로 만들 수 있습니다.
단, 프로젝트 클립 리스트 창에서 컨텍스트 메뉴인 '클립 내보내기'를 실행하면 선택된 클립만 내보내기 할 수 있습니다.
