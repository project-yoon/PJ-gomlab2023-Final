미리 보기 패널

미리보기 패널은 프로젝트 클립 리스트에 등록된 클립의 원본 콘텐츠(효과가 적용되지 않은 상태)를 미리 보여주거나 타임라인 작업 영역에 배열된 클립들의 결과물을 미리 보여주는 모니터 화면입니다. 아울러 오브젝트 클립 및 이펙트 적용할 때 보이는 프레임 컨트롤을 배치, 조절하는 편집 작업 영역으로도 사용됩니다.

미리보기 패널은 메모리의 효율 등을 감안해서 설정된 해상도보다 저해상도로 보여주기에 영상이 거칠게 보일 수 있습니다. 인코딩을 하면 실제 설정된 해상도로 만들어 집니다.

미리보기 패널은 크게 타임라인 모드와 클립 모드를 제공하며 편집 상태에 따라 자동으로 모드가 전환됩니다.

1. 타임라인 모드

타임라인의 시간에 맞춰 편집된 전체 클립을 처음부터 끝까지 보여주는 모드입니다. 타임라인 작업 영역에 포커스가 있을 때 전환됩니다. 타임라인 모드에서는 타임라인 패널 상단에 붉은색 구분선이 나타납니다.

2. 클립 모드

프로젝트 클립 리스트 창에 모아 놓은 클립을 원본 그대로 모니터할 수 있는 모드입니다. 소스 및 클립 효과 패널에서 프로젝트 클립 리스트의 클립이 선택되었을 때 전환됩니다. 클립 모드에서는 프로젝트 패널 상단에 붉은색 구분선이 나타납니다.

클립 모드는 편집 상태에 따라 소스 클립 모드, 가상 클립 모드, 트랙 클립 모드가 있습니다.

(1) 소스 클립 모드

프로젝트 클립 리스트 창에 있는 소스 클립을 원본 그대로 보여 줍니다.

- 소스 클립 모드 수동 전환 상태

편집 클립만 보입니다.

(2) 가상 클립 모드

텍스트 클립 또는 오브젝트 클립의 속성 편집 시 타임라인 Seekbar 기준의 화면을 보며 클립을 편집하는 클립 모드입니다.

텍스트 클립 또는 오브젝트 클립을 타임라인 Seekbar 기준의 화면을 보며 편집하도록 자동으로 전환됩니다. 모니터 창에 이 나타나고 타임라인 패널의 재생 헤드는 클립의 맨 앞으로 이동되면서 비활성 상태 가 됩니다.

가상 클립 모드에서 재생 헤드 이동은 미리보기 패널의 '클립 재생 헤드' 또는 '키프레임 컨트롤 재생 헤드'를 사용하십시오. 타임라인 패널을 클릭하면 작업 중인 속성 편집창이 닫히고 설정값이 클립에 적용됩니다. 타임라인 모드 전환 버튼을 클릭하면 타임라인 모드 로 바뀌면서 타임라인의 모든 클립이 모니터 화면에 보입니다. 버튼을 누르면 다시 가상 클립 모드로 전환됩니다.

(3) 트랙 클립 모드

Seekbar를 기준으로 타임라인에 있는 클립에 이펙트를 적용하거나 수정할 때 전환됩니다. 상단 트랙의 클립으로 화면이 가려지는 것을 방지하고 하단 클립만 화면을 보면서 이펙트를 적용하거나 수정합니다.

트랙 클립 모드에서는 모니터 창에 타임라인 모드 전환 버튼이 나타나고 타임라인 패널의 재생 헤드는 클립의 맨 앞으로 이동되면서 비활성 상태 가 됩니다.

트랙 클립 모드에서 재생 헤드 이동은 미리보기 패널의 '클립 재생 헤드' 또는 '키프레임 컨트롤 재생 헤드'를 사용하십시오. 타임라인 패널을 클릭하면 작업 중인 속성 설정창이 닫히고 설정값이 클립에 적용됩니다.
버튼을 누르면 타임라인 모드 로 바뀌면서 재생 헤드가 위치한 트랙의 모든 클립이 모니터 화면에 보입니다.
버튼을 다시 클릭하면 트랙 클립 모드로 전환되며 재생 헤드에 위치한 모든 트랙의 클립이 보입니다.
