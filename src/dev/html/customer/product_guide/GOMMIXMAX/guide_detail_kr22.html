타임라인 패널

트랙 컨트롤 영역

트랙 메뉴
트랙 메뉴 비디오/오디오 트랙을 추가하거나 삭제하는 팝업 메뉴 버튼입니다. 팝업 메뉴는 비디오 트랙 추가, 오디오 트랙 추가, 비디오 트랙 삭제, 빈 트랙 삭제, 트랜지션 트랙 ON/OFF 메뉴가 있습니다.
빈 트랙 삭제 메뉴는 사용하지 않는 빈 트랙을 삭제하는 기능입니다.
단 마스터 트랙은 삭제되지 않습니다.
타임라인 눈금자
타임라인 눈금자 시간 눈금자를 보여줍니다. 타임라인 확대/축소 버튼을 눌러서
타임라인 시간 단위를 늘리거나 좁혀서 볼 수 있습니다. 길이가 긴 클립이 입력되었을 때 시간대에 맞게 눈금자가 자동으로 변경되기도 합니다.
트랙 컨트롤 영역
클립 자르고 덮어쓰기 모드 와 클립 자르고 삽입 모드 클릭을 통해 지정 및 변경합니다.
트랙의 높이를 줄이거나 원래 높이로 변경합니다.
트랙의 이름을 보여줍니다. 트랙 컨트롤 영역에서 마우스 오른쪽 버튼을 누르면 '트랙 이름 변경' 메뉴에서 이름을 변경합니다. 마스터 V1트랙 은 지워지지 않으며, V2 트랙부터 이름 변경, 삭제가
가능합니다.
선택된 트랙의 비디오를 보거나 감춥니다.
선택된 트랙의 소리를 음소거하거나 해제합니다.
선택된 트랙을 잠급니다. 잠겨진 트랙에서는 클립을 선택, 이동, 삽입, 삭제 등을 할 수 없습니다.
[트랙 트랜지션] 트랙을 열거나 접습니다.
[트랙 트랜지션] 트랙이 나타납니다.
타임라인의 시간 단위 눈금자를 확대/축소합니다.
확대/축소 시 편집 길이도 같이 확대/축소됩니다.
타임라인에 넓게 배치된 클립들을 현재 창의 너비에 맞춰서 봅니다.
