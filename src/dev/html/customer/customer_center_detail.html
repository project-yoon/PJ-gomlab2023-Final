제품 설치

무료 회원의 경우 곰믹스 맥스 다운로드 시 제공되는 유상 폰트 사용이 가능한가요?

회원 무료 정책의 경우 유상 폰트는 체험만 가능합니다.
유상 폰트를 포함하여 편집 시 알림 팝업이 뜹니다.
 

 
[체험하기] 클릭 시 유상 폰트를 적용할 수 있으나, 내보내기 시 유상 폰트는 제외하고 내보내기 됩니다.
유상 폰트를 포함해 영상을 내보내고 싶다면 정품 구매 페이지에서 구매를 진행해주세요.
 
구독 취소자의 경우 유상 폰트가 적용된 기존 프로젝트에서 내보내기 시 비회원과 동일하게 유상 폰트를 제외하고 내보내기 됩니다.
